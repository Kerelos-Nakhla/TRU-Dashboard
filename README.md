# 📊 Transaction Data Analysis Dashboard (Excel)

## 📌 Overview
This project presents an interactive Excel dashboard designed to analyze transaction data and extract meaningful business insights.

The dashboard enables users to explore data across multiple dimensions such as categories, products, users, payment methods, and locations.

---

## 🎯 Objectives
- Analyze transaction behavior
- Identify top-performing categories and products
- Track revenue, fees, and cashback
- Understand user and merchant activity
- Provide actionable business insights

---

## 📊 Dashboard Features

### 🔹 KPIs
- Total Revenue
- Total Fees
- Total Cashback
- Number of Transactions
- Number of Categories
- Number of Products
- Number of Users
- Number of Merchants
- Cashback Ratio
- Fee Ratio

### 🔹 Analysis Sections
- Transactions by Category & Product
- Revenue by Category & Product
- Transactions by Payment Method
- Transactions by Device Type
- Revenue by Location
- Monthly Trends (Revenue, Users, Transactions)
- Top Users & Top Merchants

---

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning Techniques

---

## 📈 Key Insights
- Streaming and utility-based categories drive the highest revenue
- Urban areas contribute the majority of revenue
- Android dominates device usage
- Bank Transfer is the most used payment method
- Stable monthly revenue trend with slight seasonal variation
